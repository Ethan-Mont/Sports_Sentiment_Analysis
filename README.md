# Sports_Sentiment_Analysis
An intermediate Machine Learning project focused on predicting the outcome of NFL games. This project first establishes a baseline model that is selected from a variety of options. We then obtain online sentiment scores using reddit data and augment the baseline model with these sentiment scores. The resulting two-stage model performs significantly better during training, but testing scores remain similar to baseline estimates.
